ANESTEZJOLOGIA - PWA
Autorstwo: Tech.Wet Woroch Marcin

CO ZNAJDZIESZ W PACZCE
- index.html - glowny ekran aplikacji
- styles.css - wyglad aplikacji
- app.js - logika zapisu, historii pacjentow, PIN, kopii zapasowej i wydruku
- manifest.json - plik potrzebny do instalacji jako PWA
- sw.js - service worker do dzialania offline
- vercel.json - ustawienia dla Vercel
- folder icons - ikony aplikacji

JAK WDROZYC NA VERCEL
1. Wejdz na vercel.com i zaloz konto lub zaloguj sie.
2. Kliknij Add New -> Project.
3. Wybierz zakladke do importu projektu lub przeciagij caly folder anestezjologia-pwa.
4. Przy wdrozeniu nie musisz ustawiac build command ani frameworka. To jest statyczna aplikacja.
5. Kliknij Deploy.
6. Po chwili dostaniesz publiczny link.

JAK ZAINSTALOWAC NA ANDROIDZIE
1. Otworz publiczny link w Chrome na telefonie.
2. Kliknij menu przegladarki.
3. Wybierz Dodaj do ekranu glownego.
4. Otworz aplikacje z ikony.

DANE I PRYWATNOSC
- Historia pacjentow zapisuje sie lokalnie na urzadzeniu.
- Dane NIE znikaja po zamknieciu aplikacji ani po restarcie telefonu.
- Dane moga zniknac po wyczyszczeniu danych przegladarki lub resecie telefonu.
- Dlatego warto regularnie robic Eksport kopii.
